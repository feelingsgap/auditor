최종 목표물 : 연도별/과목별 문제지를 html 로 정리하자.
과정 :
1. pdf 에서 연도별 문항을 파악해서 크롭할 위치정보와 문항을 json 으로 추출한다.(연도별 문항수는 120문제)
 /auditor/{year}/{year}_data.json
2. json 의 정보로 연도별 문항을 png 파일로 크롭한다.
 /auditor/{year}/question/{year}_*.png
3. pdf 에서 연도별 페이지를 png 파일로 캡쳐한다.
 /auditor/{year}/page/{year}_{page}.png
4. 연도별 json 의 정보로 문항과 그에 따른 페이지,크롭 이미지를 html 로 정리한다.
 /auditor/index.html

주의사항 :
- pdf 1페이지는 문항과 관련없는 유의사항이라서 캡처와 분석에서 건너띈다.
- 시험지의 크기는 년도별로 다르다.
- 시험지의 문항 구성은 좌우 단락으로 구분되어, 좌에서 우로 순서가 있다.
- 문항의 높낮이는 상이하다.
- 전체 문항의 시작좌표를 순차적으로 검증해서 구한다음에, 문항의 종료 좌표를 구한다.

# 문항의 분류와 순서
1. 감리 및 사업관리(25)
2. 소프트웨어 공학(25)
3. 데이터베이스(25)
4. 시스템 구조(25)
5. 보안(20)
