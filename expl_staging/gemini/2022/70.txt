정답: ②

**해설:**
데이터 마이닝 모델의 의사결정트리 알고리즘(`CART` 등)에서 데이터 분할 시 평가 지표로 사용하는 노드의 불순도인 **Gini 계수**를 계산하는 문제입니다.
*   산출 공식: `Gini = 1 - Σ(해당 클래스 확률의 부분 제곱합)`

**계산 전개:**
*   전체 레코드 수: 클래스0 (1개) + 클래스1 (5개) = **총 6개**
*   클래스 `0`의 속함 확률 편차(p_0) = `1 / 6`
*   클래스 `1`의 속함 확률 편차(p_1) = `5 / 6`
*   Gini 불순도 = `1 - { (1/6)^2 + (5/6)^2 }`
*   = `1 - (1/36 + 25/36)`
*   = `1 - 26/36`
*   = `10/36` = `0.27777...`

따라서 Gini 계수를 반올림하면 약 **0.278**의 불순도라는 지표를 띠게 됩니다.
