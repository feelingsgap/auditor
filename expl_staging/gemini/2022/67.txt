정답: ③

**해설:**
Apriori 연관분석(Association Rules) 과정에서 도출되는 규칙 **X → Y 의 신뢰도(Confidence 확률)** 를 계산하는 문제입니다.
*신뢰도 수식: `Confidence(X → Y) = (X와 Y가 동시에 포함된 거래수) / (X를 포함하는 거래수)`*

**단일/조합 지지도(발생수) 파악:**
*   A 포함: (1, 2, 3, 5) -> 4건
*   B 포함: (1, 3, 4) -> 3건
*   C 포함: (1, 5, 6) -> 3건
*   (A∩B) 포함: (1, 3) -> 2건
*   (A∩C) 포함: (1, 5) -> 2건

**각 보기별 신뢰도 계산:**
*   ① **A → B:** (A∩B 출현) 2건 / (A 명제 조건 출현) 4건 = `0.50 (50%)`
*   ② **A → C:** (A∩C 출현) 2건 / (A 명제 조건 출현) 4건 = `0.50 (50%)`
*   ③ **B → A: (A∩B 출현) 2건 / (B 명제 조건 출현) 3건 = `0.667 (약 66.7%)`** (가장 높음!)
*   ④ **B → C:** (B∩C 출현) 1건 / (B 출현 조건) 3건 = `0.33 (약 33.3%)`

따라서 발생 신뢰도(확률적인 강도)가 가장 우세한 규칙은 **B → A** 입니다.
