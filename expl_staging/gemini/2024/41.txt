정답: 1 (①)

**해설:**
Python 리스트의 대표적인 3가지 메서드 동작 메커니즘을 묻는 문제입니다. 코드를 한 줄씩 디버깅해 봅니다.

**상세 분석:**
* 초기 리스트: `language = ['python', 'java', 'go', 'ruby', 'c']`
* `pop = language.pop()` ➔ 인덱스를 비우면 맨 끝을 날립니다. `pop`엔 `'c'`가 담기고 리스트에서 `'c'`가 날아갑니다.
* `pop2 = language.pop(1)` ➔ 이번엔 1번 방을 날립니다. 0번 `python` 다음인 `'java'`가 뽑혀 나옵니다. (현재 리스트: `python`, `go`, `ruby`)
* `language.insert(1, pop)` ➔ 1번 자리에 아까 킵해둔 `'c'`를 억지로 쑤셔 넣습니다. 뒤의 애들은 한 칸씩 밀립니다. (현재: `python`, `c`, `go`, `ruby`)
* `language.insert(2, pop2)` ➔ 2번 자리에 `'java'`를 다시 쑤셔 넣습니다. (현재: `python`, `c`, `java`, `go`, `ruby`)
* 여기까지 리스트는 원래 멤버 그대로 순서만 뒤섞인 채 부활했습니다.
* `print(language.index('go'))` ➔ `'go'`가 현재 몇 번 방에 밀려났는지 찾습니다. 0번(`python`), 1번(`c`), 2번(`java`), **3번방(`go`)**. 따라서 출력은 **3**.

**📌 메서드 요약:**
* `.pop()` : 맨 뒤 뽑기, `.pop(i)` : 위치 값 뽑아냄.
* `.insert(i, 찾을값)` : 해당 인덱스에 밀어 넣기.
* `.index(찾을값)` : 해당 값이 위치한 번호를 스캔.
