정답: 4 (④)

**해설:**
패턴 분류 기계학습 모델 중 나이브 베이즈(Naive Bayes) 이론의 핵심인 "거꾸로 역추적 사후 확률 추정"을, 확률론의 꽃인 분수 형태 베이즈 정리 기본기 수학 공식 도구 폼에 일치하게 대입시켜 수기로 사칙 연산 결괏값을 똑 떨어지게 연산해 맞추어내는 응용문제입니다.

**상세 분석:**
* **내가 구하고 밝혀내고자 하는 궁극의 것 최종 지령:** 어떤 미지의 특정 X라는 특성 단서 발작(증상 현상 증거)이 내 눈앞에서 발현 딱 관측 목격되었을 때(조건), 그럼 이 녀석이 대체 여러 우주 중에 C1 세계관 클래스 우주(결론 집단) 부류에 속한 놈일 그 확률의 퍼센티지 한계망이 교집합 상 대충 얼마인가? `P( C1 | X )` 를 계산하여 도출하라. (이걸 조건부 역산 확률, 혹은 사후 확률이라 칭함).
* **베이즈 정리 수학 절대 공식 전개 스니펫 분자/분모 폼:** 
  `P( C1 | X, 사후확률! )= [ P(X | C1, 증상 관측 단서) ×  P(C1, 베이스 사전확률) ]  / (나누기)  P( X 우주 전체가 포착될 통계적 모든 지분 총확률 합계) 전체 확률`
* 지문 데이터 해부 인입:
  * `P(C1) 무지성 사전확률 기반 베이스` = 0.5 (현실 세계 절반 50% 확률)
  * `P(C2) 무지성 사전확률 기반 베이스` = 0.5 (현실 세계 절반 50% 확률)
  * `P(X | C1) C1 클래스 안쪽 그물망에서 X현상이 나올 순수 단서 발생률` = 0.6 
  * `P(X | C2) 다른 라이벌 C2 클래스 진영에서 X현상이 우연히 겹쳐 나올 순수 단서 발생률` = 0.4
* 1단계 산술 풀이 진행: **역산 전체 하단 분모 바탕 베이스 판떼기인 `P(X)` 구하기 (세상 모든 C1, C2 등등의 파생 우주 진영 파편에서 어쨌든 저쨌든 X라는 증상 결괏값이 터져 등장할 그 모조리 긁어모은 총확률 빈도수). (전체 확률의 법칙 법칙)**
  * (C1 우주 기본 점유율 * 거기서 X 등장률 곱) 더하기 + (C2 우주 기본 점유율 * 거기서 X 등장률 곱)
  * 수치 치환 대입: (0.5 × 0.6) 더하기 + (0.5 × 0.4) = 앞 덩어리 0.3 + 뒷 덩어리 0.2 = 도합 **0.5 (50% 베이스 판떼기 완성)**
* 2단계 최종 본 게임 역산 나눗셈 공식 대입.
  * **[위] 상단 Numerator 분자:** 오직 오리지널 C1 우주 진영 세계 속에서 발생한 X 퓨어 발현 확률 교집합 ➔ (0.5 × 0.6) = **0.3 순도 추출**
  * **[밑] 하단 Denominator 분모 바탕 통치기:** 아까 위 1단계에서 피똥 싸게 구한 증상 모든 발현 총 확률 세계 ➔ **0.5 지분 바운더리**
* 도출 계산: 0.3 파이 조각 / (나누기) 0.5 전체 원 파이 = **0.6 (즉 60% 비중의 사후 환산 확률 당첨!)**

**📌 핵심 요약 수식 대입 공식:**
* 베이즈 확률론 정리 P(과녁 정답지 | 관측된 단서) = 상단 분자 지붕(해당 타겟 과녁 순수 사전확률 베이스 * 해당 과녁 안에서의 단서 출현율 곱집합) / 하단 밑판 분모 베이스(세상 모든 제3의 과녁 잡탕들의 단서 출현율 확률 곱셈 치환 총합 누적 더하기). 
* 수학 도출식 공식의 암산 직관화 축약 폼: 상단 수치 (0.5 * 0.6) / 나누기 전체 괄호 하단 ( 앞단 C1: (0.5 * 0.6) + 뒷단 C2 제3변수: (0.5 * 0.4) ) = 0.3 / 0.5 비율 도출로 답은 0.6
