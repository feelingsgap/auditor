정답: 4 (④)

**해설:**
JAVA 바이트 클래스 파일 코드 구문 자체를 실행 없이 뜯어 스캐닝 분해하여 시큐어코딩(보안 약점 47개 가이드 등)을 취약하게 위반했거나, 개발자가 실수한 어이없는 Null Pointer 역참조 같은 프로그래밍 논리 버그를 짚어서 찝어내주는 오픈소스 정적 감식 툴(SAST 소프트웨어) 이름을 골라냅니다.

**상세 분석:**
* **① ModSecurity (모드 시큐니티):** 이 녀석은 뒤쪽 앱 서버의 Java 소스코드를 보는 게 아니라 최전방 실제 웹서버(Apache나 Nginx 등) 앞단에 붙어서 실시간으로 유저가 날리는 악성 해킹 텍스트 페이로드를 스캔해 쳐내는 웹 필터 트래픽 방패인 **WAF (웹 방화벽)** 모듈 플러그인입니다. 멈춰있는 소스코드 정적 분석 불가.
* **② OpenVAS (오픈바스):** 서버 IP 포트를 블랙박스 상태에서 마구 찌르고 원격 스캐닝 포트 스캔을 돌려 열려있는 취약점 홀이나 OS 구버전을 진단하는 네트워크 외부 동적 인프라 점검 스캐너 도구 인프라입니다. 자바 프로그램 소스를 분석하지 않습니다.
* **③ OllyDbg (올리디버거):** 해킹과 리버싱의 조상격 교과서 바이블로 불리는 C/C++ x86 어셈블리어 레지스터 타겟 전용 윈도우 동적 런타임 리버싱 디버깅 크랙 테스터. 자바 웹 코드랑은 거리가 아주 멉니다.
* **④ SpotBugs (정답 / 낭중지추의 유일 툴):** 
  * 과거 찬란히 이름을 날리던 전설의 구(舊) 자바 정적툴 **FindBugs** (파인드벅스) 프로젝트의 직계 계승자(포크 버전 최신 툴) 프로젝트명입니다. 
  * 자바 컴파일이 완료된 `.class` 파일의 바이트코드 시그니처 정적 흐름 패턴 등을 수학적으로 맵핑 후 추적 스캔해, 개발자가 뺀 널포인터 예외, 무한루프 레이스, 그리고 XSS 따위의 보안 약점 위반 사항 목록 위치를 쫘아악 엑셀 리포트와 점수로 뽑아주는 가장 대표적인 공개 자바 정적 SW 분석(SAST) 툴입니다. 

**📌 바이트 심사 핵심 요약:**
* 자바 전용 JVM 바이트코드 정적 보안 취약점 패턴 분석 전용 SW: **과거 전설 FindBugs ➔ 현재는 SpotBugs 스팟벅스로 계승.** (마찬가지로 PMD, SonarQube 등과 함께 시큐어 감리판에서 표준 파트너로 실무 사용됨).
