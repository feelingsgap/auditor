정답: ②

**해설:**
텍스트 마이닝 두 문서(Document)의 용어 빈도 벡터 간의 기하학적 **코사인 유사도(Cosine Similarity)**를 산출하는 수식 문제입니다. (A 벡터 · B 벡터) / (|A 크기| × |B 크기|)
*   **내적(Dot Product):** 양 문서의 동일 과일 위치값들을 위아래 곱해서 다 더합니다. 
    (3×1) + (1×1) + (0×2) + (3×1) + (1×0) + (2×2) + (1×4) + (0×3) = **15**
*   **Doc1 선분 크기:** 루트 ( 3² + 1² + 0² + 3² + 1² + 2² + 1² + 0² ) = 루트 25 = **5**
*   **Doc2 선분 크기:** 루트 ( 1² + 1² + 2² + 1² + 0² + 2² + 4² + 3² ) = 루트 36 = **6**
*   **유사도 결괏값:** 15 / (5 × 6) = 15 / 30 = **0.5**
