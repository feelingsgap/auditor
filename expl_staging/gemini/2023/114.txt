정답: 2 (②)

**해설:**
웹사이트 뚫리나 안 뚫리나 XSS, SQL 인젝션 공격 등 모의 테스트 때려보는 **웹 취약점 자동화 스캐너 / 프록시 도구(Web Vulnerability Scanner Tool)** 단톡방에서 직업군이 전혀 다른 잡상인 해커 무기를 걷어냅니다. 

**블랙리스트 조회:**
* ① Burp Suite: 전 세계 웹 해커/버그헌터 99%가 기본 장착하는 웹 진단 모의 해킹 인터셉트 웹 프록시 최강 툴.
* ③ Paros: OWASP ZAP의 조상님 격으로, Java 기반으로 만들어진 전통의 훌륭한 웹 스캐닝 및 프록시 국산차.
* ④ WebScarab: 이름부터 웹 딱지 붙음. OWASP 성님들이 제공했던 유서 깊은 웹 프록시 보안 패킷 조작툴. 
* **[넌 나가라] ② Hydra (히드라):**
  ➔ 이름만 들어도 그리스 신화 아가리 여러 개인 그놈. 이놈은 웹페이지 구조적 취약점 찾는 우아한 스캐너가 아니라, **Telnet, FTP, SSH 구멍 포트 접속망들에 "비밀번호 리스트 10만 개" 개 꼴아박으면서 될 때까지 두들겨 뚫는 무지성 계정 탈취 Brute force(무차별 대입 패스워드 크랙) 샷건 도구 전용 프로그램**입니다. 웹 애플리케이션 구조 진단 툴 카테고리로 넣기엔 분야가 너무 멉니다. 정답 ②.
