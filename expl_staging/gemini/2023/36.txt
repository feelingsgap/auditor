정답: 4 (④)

**해설:**
과거 대기업망을 10년 넘게 호령했던 XML 기반 거대 공룡 웹 통합 서비스 **SOA (Service Oriented Architecture)** 핵심 3대장 표준 통신 규격 스펙 매칭 퍼즐입니다.

**상세 분석:**
* **가. SOAP:** (단어에 Protocol이 들어감) 웹 서비스끼리 XML 편지를 주고받을 때 편지 봉투에 머리, 몸통 잘라서 넣는 **"통신 메시지 교환 표준" ➔ C 매칭**
* **나. WSDL:** (Description L) "우리 사이트 API 명세서 요약판입니다. 요런 인자 쏘면 요 데이터 리턴해줌" 설명서 **"서비스 인터페이스 정의 표준" ➔ B 매칭**
* **다. WS-BPEL:** 단위 별로 깔짝대는 API 컴포넌트들을 블록 조립해서 A끝나면 B가고 조건문 C태워라는 거대한 사내 출장 결재 파이프라인으로 엮어버리는 **"워크플로우(프로세스) 실행용 고차원 코딩 언어" ➔ A 매칭**
* 정확한 퍼즐 조립은 **가-C, 나-B, 다-A (④번)** 라인입니다.
