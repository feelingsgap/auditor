정답: 3 (③)

**해설:**
의사결정 테이블(Decision Table) 구석구석을 4개의 테스터 케이스(TC)가 찔러봤을 때, 8개의 전체 룰 기둥 중 몇 개나 중복 안 되게 커버 쳤는지(Rule Coverage 비율) 산수하는 킬 카운트 문제입니다.

**상세 분석:**
* **TC1 (X= -200, Y= 0):** C1(F), C2(F), C3(F) ➔ 세 개 다 False 나오는 기둥은 맨 오른쪽 **[Rule 8]** 맞음. (1 킬)
* **TC2 (X= 0, Y= 200):** C1(0>-100? T), C2(200>150? T), C3(0+200>150? T) ➔ 세 개 다 True 기둥은 맨 왼쪽 **[Rule 1]** 맞음. (2 킬째)
* **TC3 (X= -100, Y= 150):** C1(-100>=-100? T), C2(150>150? F), C3(-100+150=50>150? F) ➔ TFF 기둥은 **[Rule 4]** 맞음. (3 킬째)
* **TC4 (X= -100, Y= 250):** C1(T), C2(250>150? T), C3(T) ➔ TTT 기둥은 어라? TC2랑 겹치는 **[Rule 1]** 입니다. (중복 타격 무효)
* 결론: 8개의 Rule 중 4개의 어택이 고유하게 때린 건 [Rule 1, Rule 4, Rule 8] 딱 3개 기둥. 
* 룰 방어율 커버리지 = **3 / 8** (③ 정답)
