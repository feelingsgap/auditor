정답: 1 (①)

**해설:**
네트워크 서브넷 마스크 IP 쪼개기(CIDR) 실전 수학 연산 문제입니다.
* **미션:** IP `206.17.38.36 / 28` 의 이 동네 주소 대장(네트워크 대표 주소)을 찾아라.

**[서브넷 마스킹 계산 과정]**
1. `/28`의 의미: 총 32비트짜리 마스크 중 앞의 28개를 1로 다 채우고, 뒤의 4개만 0으로 남겨 호스트 집합소로 쓴다는 뜻입니다.
   ➔ 서브넷 마스크: `255.255.255.???` (앞의 24비트는 꽉 찼고, 마지막 4번째 옥텟은 `1111 0000` = 즉 십진수로 **240**)
2. **동네 주소(네트워크 주소) = 원본 IP 주소 "AND" 서브넷 마스크 조작.**
   ➔ 앞의 3덩어리 `206.17.38`은 어차피 마스크가 255니까 백날 AND 연산해 봐야 그대로 프리패스.
   ➔ 승부는 4번째 쪼가리, **`36`이라는 IP 번호와 마스크 `240`의 이진수 AND 연산**입니다.
3. 이진수 배틀:
   - 36 =  `0 0 1 0 0 1 0 0`
   - 240 = `1 1 1 1 0 0 0 0`
   --------------------------- (둘 다 1일 때만 1 떨어지는 AND 연산)
   - 결과= `0 0 1 0 0 0 0 0`   ➔ 이걸 십진수로 복원하면 32 !!
4. 따라서 이 동네의 맨 앞 네트워크 대문 주소는 꼬다리에 32가 박힌 **① 206.17.38.32** 로 귀결됩니다.
