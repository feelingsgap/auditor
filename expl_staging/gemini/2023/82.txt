정답: 4 (④)

**해설:**
C 프로그래밍 언어의 뼈대인 **아스키코드(ASCII) 16진수 문자 연산** 기본기입니다.

**상세 분석:**
* 문제에서 친절하게 문자 `'0'` 의 메모리 2진수가 `0011 0000` 이라고 뿌려줬습니다. 이를 16진수로 끊으면,
  - `0011` = 3
  - `0000` = 0
  - 즉, **문자 '0' 의 16진수 값은 `0x30`** 입니다. (외워두면 피가 되고 살이 됨)
* 그럼 우리가 구해야 할 **문자 `'9'`** 의 값은?
  ➔ 아스키코드는 0부터 9까지 차례대로 번호표를 뽑아놨으므로, '0'에서 손가락을 9번 튕기면 됩니다.
  ➔ `0x30 + 9` = **`0x39`** 
* (상식 주의점: `int C = 9;` 였다면 값은 `0x09` 지만, 따옴표가 붙은 `char C='9';` 이므로 화면 출력용 문자 코드인 `0x39` 가 들어갑니다.) 정답 ④번.
