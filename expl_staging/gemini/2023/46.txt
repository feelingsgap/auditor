정답: 3 (③)

**해설:**
Java 진영의 소스코드 고문 도구 **JUnit4** 의 어노테이션(Annotation) 폭죽 터지는 발동 순서 맞추기 체력장입니다.

**상세 분석:**
* **폭격 순서 율법서:**
  1. `@BeforeClass` (클래스 뜰 때 딱 1번) ➔ DB 연결, 웹서버 올리기. (**a 가동**)
  2. `@Before` (각 테스트 직전에 매번 빵!) ➔ 변수 초기화. (**c 가동**)
  3. `@Test` (테스트 1번) ➔ **t1 발사!**
  4. `@After` (테스트 1번 직후 매번 빵!) ➔ 변수 비우기. (**d 가동**)
  5. --- 테스트 2번 t2가 남았으므로 2,3,4번 반복 ---
  6. `@Before` ➔ (**c 가동**)
  7. `@Test` ➔ **t2 발사!**
  8. `@After` ➔ (**d 가동**)
  9. `@AfterClass` (모든 테스트가 끝나고 지구 멸망 전 딱 1번) ➔ DB 해제. (**b 가동**)
* **최종 타임라인:** **a ➔ c ➔ t1 ➔ d ➔ c ➔ t2 ➔ d ➔ b** 
* 정답은 이 박자를 오차 없이 타고 있는 **③번** 입니다.
