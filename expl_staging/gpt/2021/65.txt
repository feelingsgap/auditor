정답: ② (2번)

해설:
상사가 삭제되면 Super_ENO는 SET NULL이고, 소속 부서 삭제는 SET DEFAULT, 부서 관리자의 삭제도 SET DEFAULT다.
따라서 상사가 삭제되면 디폴트 상사로 배치된다는 설명은 틀리다.

각 보기 분석:
① 부서 삭제 시 디폴트 부서로 배치 - 맞다.
② 상사 삭제 시 디폴트 상사로 배치 - 틀리다. 정답이다.
③ 부서번호 변경 시 소속 부서번호도 갱신 - 맞다.
④ 관리자 삭제 시 디폴트 관리자로 배치 - 맞다.

핵심 정리:
SET NULL과 SET DEFAULT를 구분해야 한다.
