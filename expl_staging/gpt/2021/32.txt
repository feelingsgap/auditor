정답: ② (2번)

해설:
static 메서드에서는 인스턴스 변수나 인스턴스 메서드를 직접 사용할 수 없다.
따라서 count 참조와 static 메서드 호출만 가능하다.

각 보기 분석:
(가) a 참조 - 인스턴스 변수라서 오류다.
(나) count 참조 - static 변수라서 가능하다.
(다) getA() 호출 - 인스턴스 메서드라서 오류다.
(라) getCount() 호출 - static 메서드라서 가능하다.

정답 분석:
① 가, 다 - 오류가 발생하는 줄이라 틀리다.
② 나, 라 - 오류가 발생하지 않는 줄만 묶인 정답이다.
③ 가, 나, 다 - 가와 다가 오류다.
④ 가, 나, 다, 라 - 가와 다가 오류다.

핵심 정리:
static 문맥에서는 static 멤버만 직접 접근할 수 있다.
