정답: ④ (4번)

해설:
TMMi 레벨 2의 테스트 계획 프로세스 영역에 속하지 않는 세부 목표를 묻는 문제다.

제품 위험 평가, 테스트 접근 방식 설정, 테스트 추정치 설정은 테스트 계획 영역의 핵심 활동이다. 반면 테스트 성능 지표 설정은 상위 수준의 측정·통제 관점과 더 관련이 있다.

각 보기 분석:
① 제품 위험 평가 수행 — 테스트 계획 영역에 해당한다.
② 테스트 접근 방식 설정 — 테스트 계획 활동이다.
③ 테스트 추정치 설정 — 테스트 계획 활동이다.
④ 테스트 성능 지표 설정 — 테스트 계획의 세부 목표로 보기 어렵다. 정답이다.

핵심 정리:
TMMi 레벨 2의 테스트 계획은 위험, 접근법, 추정치 설정 중심이며 성능 지표는 직접 항목이 아니다.
