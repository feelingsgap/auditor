정답: ③ (3번)

해설:
획득가치 분석(EVM, Earned Value Management)에서 주어진 수치(EV=800, PV=900, AV=700)를 이용해 성과지수와 성과차이를 계산하는 문항이다. 단, 문제에서 AV는 실제원가(AC, Actual Cost)를 의미한다.

주요 EVM 공식:
- 일정 성과차이(SV) = EV - PV = 800 - 900 = -100
- 원가 성과차이(CV) = EV - AC = 800 - 700 = +100
- 일정 성과지수(SPI) = EV / PV = 800 / 900 ≈ 0.89 (1보다 작다)
- 원가 성과지수(CPI) = EV / AC = 800 / 700 ≈ 1.14 (1보다 크다)

각 보기 분석:
① 일정 성과차이(SV)는 +100이다 — SV = EV - PV = -100이므로 틀림.
② 원가 성과차이(CV)는 -100이다 — CV = EV - AC = +100이므로 틀림.
③ 일정 성과지수(SPI)는 0보다 크다 — SPI = 0.89로 0보다 크다. 정답.
④ 원가 성과지수(CPI)는 1보다 작다 — CPI = 1.14로 1보다 크다. 틀림.

핵심 정리:
SPI < 1: 일정 지연 / SPI > 1: 일정 앞당김.
CPI < 1: 예산 초과 / CPI > 1: 예산 절감.
이 프로젝트는 일정은 지연(SPI<1)되었으나 비용은 절감(CPI>1) 중인 상태이다.
