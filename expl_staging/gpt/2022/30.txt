정답: ② (2번)

해설:
이진 탐색(Binary Search) 알고리즘을 구현한 Java 코드에서 배열 {1, 2, 3, 4}와 k=3으로 search()를 호출했을 때의 출력값을 찾는 문항이다.

코드 실행 추적:
- list = [1, 2, 3, 4] (인덱스 0~3)
- k = 3, left = 0, right = 3, result = -1

1회 반복 (left=0, right=3):
  mid = 0 + (3-0)/2 = 1
  list.get(1) = 2
  2 > 3? 아니오
  2 == 3? 아니오
  else: left = mid+1 = 2

2회 반복 (left=2, right=3):
  mid = 2 + (3-2)/2 = 2
  list.get(2) = 3
  3 > 3? 아니오
  3 == 3? 예 → result = mid = 2, right = mid-1 = 1

3회 (left=2 > right=1): 반복 종료

return result = 2

각 보기 분석:
① 1 — 인덱스 1(값 2)에 해당하며 오답.
② 2 — 값 3이 위치한 인덱스 2. 정답.
③ 3 — 값 3 자체이나, 코드는 인덱스를 반환한다.
④ 4 — 해당 없음.

핵심 정리:
이 이진 탐색 코드는 값을 찾았을 때 해당 인덱스(mid)를 result에 저장하고 right를 줄여 계속 탐색한다.
k=3의 위치는 인덱스 2이므로 반환값은 2.
