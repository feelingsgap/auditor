정답: ③ (3번)

해설:
데이터베이스 암호화 기술 설명 중 틀린 것을 찾는 문제다.

보기 ③은 Plug-In 방식을 설명하면서 실제로는 API 방식의 특징을 말하고 있다. 애플리케이션 서버에 라이브러리 형태 모듈을 설치하고 프로그램에서 호출하는 것은 보통 API 방식이다.

각 보기 분석:
① Plug-In 방식은 DB 서버에 모듈을 설치해 응용프로그램 수정 부담을 줄일 수 있으므로 맞다.
② API 방식은 DB 서버 영향은 적지만 응용프로그램 수정이 필요할 수 있으므로 맞다.
③ Plug-In 방식을 애플리케이션 서버 라이브러리 호출 방식으로 설명했는데 이는 API 방식에 가까워 틀리다. 정답이다.
④ 하이브리드 방식은 두 방식의 장점을 조합해 사용할 수 있으므로 맞다.

핵심 정리:
API 방식은 애플리케이션 수정이 필요하고, Plug-In 방식은 DB 서버에 모듈을 설치하는 형태다.
