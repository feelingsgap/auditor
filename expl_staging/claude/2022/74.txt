정답: ③ (3번)

해설:
혼동행렬(confusion matrix)에서 TP=45, FN=20, FP=5, TN=30이 주어졌을 때 각 성능척도를 공식에 따라 계산하여 오류를 찾는다.

혼동행렬 값:
TP(True Positive) = 45, FN(False Negative) = 20
FP(False Positive) = 5, TN(True Negative) = 30

각 보기 분석:
① 정밀도(precision) = TP / (TP + FP) = 45 / (45 + 5) = 45/50 — 공식과 값 일치. 옳다.
② 재현율(recall) = TP / (TP + FN) = 45 / (45 + 20) = 45/65 — 공식과 값 일치. 옳다.
③ 특이도(specificity) = 20/50 — 틀렸다. 특이도 공식은 TN / (TN + FP) = 30 / (30 + 5) = 30/35이다. 20/50은 FN/(TP+FP)로 어떤 표준 지표와도 일치하지 않는다.
④ 정확도(accuracy) = (TP + TN) / 전체 = (45 + 30) / (45 + 20 + 5 + 30) = 75/100 — 공식과 값 일치. 옳다.

핵심 정리:
정밀도: TP/(TP+FP) — 양성 예측 중 실제 양성 비율.
재현율(민감도): TP/(TP+FN) — 실제 양성 중 감지된 비율.
특이도: TN/(TN+FP) — 실제 음성 중 올바르게 음성으로 예측된 비율.
정확도: (TP+TN)/(TP+TN+FP+FN) — 전체 예측 중 옳은 예측 비율.
