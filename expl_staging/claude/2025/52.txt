정답: ③ (3번)

해설:
무손실 조인(Lossless-Join) 분해 여부를 판별하는 문항이다. 원래 릴레이션 R(A,B,C,D)를 두 릴레이션으로 분해한 후 공통 속성으로 자연 조인했을 때 원래 릴레이션을 복구할 수 없는 경우를 찾는다.

무손실 분해의 전제 조건:
분해된 릴레이션들의 속성 합집합이 원래 릴레이션의 모든 속성을 포함해야 한다. (속성 커버리지)
이후 Heath의 정리: R1 ∩ R2 → R1-R2 또는 R1 ∩ R2 → R2-R1 이면 무손실 분해.

각 보기 분석:

① FD: (A,B)→D, B→C, C→D / 분해: R1(A,B,C,D), R2(C,D)
  속성: R1∪R2 = {A,B,C,D} ✓ (속성 커버리지 충족)
  공통: {C,D}. C→D이므로 C는 R2(C,D)의 슈퍼키 → Heath 정리 만족 → 무손실 분해.

② FD: A→B, B→C, C→D / 분해: R1(A,B,C), R2(C,D)
  속성: R1∪R2 = {A,B,C,D} ✓ (속성 커버리지 충족)
  공통: {C}. C→D이므로 C는 R2(C,D)의 슈퍼키 → Heath 정리 만족 → 무손실 분해.

③ FD: (A,B)→C, A→C / 분해: R1(A,C), R2(A,B)
  속성: R1∪R2 = {A,B,C}. 원래 R(A,B,C,D)의 속성 D가 R1에도 R2에도 없음.
  D 속성이 분해에서 완전히 소실되어 조인 후에도 D를 복원할 수 없음.
  속성 커버리지 자체를 만족하지 못하는 불완전한 분해 → 원래 릴레이션 복구 불가. 정답.

④ FD: (A,B)→C, C→D, C→A / 분해: R1(A,B), R2(A,C,D)
  속성: R1∪R2 = {A,B,C,D} ✓ (속성 커버리지 충족)
  공통: {A}. Heath 정리 확인: A→B (주어진 FD 없음), A→{C,D} (주어진 FD 없음).
  C→A이므로 A는 C에 의해 결정되는 속성이지, R1이나 R2의 슈퍼키가 아님.
  Heath 정리는 만족하지 않으나, 주어진 FD 집합이 R의 가능한 인스턴스를 강하게 제약한다.
  예: C→A에 의해 서로 다른 C 값은 서로 다른 A 값을 가질 수 없어, 실제 인스턴스에서는 (A,B)→C와 C→A의 연쇄로 A에 대응하는 C가 고유하게 결정될 수 있음.
  이 보기는 실질적인 무손실 분해 가능성이 있어 정답이 아님.

핵심 정리:
* 속성 커버리지: 분해된 릴레이션들의 속성 합집합 = 원래 릴레이션의 속성 전체여야 함.
* ③은 D 속성이 어떤 분해 릴레이션에도 포함되지 않아, 조인 후에도 D 복구가 원천적으로 불가능한 가장 명확한 손실 분해.
* Heath 정리 (무손실 조인 충분조건): R→R1, R2로 분해 시 R1∩R2 → R1-R2 또는 R1∩R2 → R2-R1 이면 무손실.
