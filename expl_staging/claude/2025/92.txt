정답: ① (1번)

해설:
CRC 하드웨어 회로 다이어그램에서 생성 다항식을 유도하는 문항이다.

CRC 회로 분석 방법:
- 직렬 shift 레지스터의 개수 = 다항식의 최고 차수
- XOR(Exclusive OR) 피드백 연결 위치 = 다항식의 항

회로 구성 (이미지 분석):
4개의 shift 레지스터가 직렬 연결됨 → 최고 차수 = x^4 (상수항 포함)
XOR 연결이 레지스터 2번 위치(0-indexed 기준 x^2에 해당)에 있음 → x^2 항 포함
피드백 입력 연결 → 상수항 1 포함

생성 다항식: x^4 + x^2 + 1

각 보기 분석:
① x^4 + x^2 + 1 — 4개 레지스터 + x^2 위치 XOR. 회로와 일치. 정답.
② x^2 + 1 — 레지스터 2개에 해당. 4개 레지스터 회로와 불일치.
③ x^4 + x^3 + x — x^3, x 항 위치에 XOR이 있어야 함. 회로와 불일치.
④ x^3 + x — 3개 레지스터 + x^1 위치. 회로와 불일치.

핵심 정리:
CRC 다항식 읽는 방법:
- 레지스터 개수 n → x^n 포함
- i번째 레지스터 출력에서 XOR 피드백 → x^i 항 포함
- 항상 상수항(x^0 = 1) 포함
대표 CRC 표준: CRC-16 = x^16+x^15+x^2+1, CRC-32 = x^32+x^26+...+x^1+1
