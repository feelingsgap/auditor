정답: ② (2번)

해설:
R과 S의 완전 외부 조인(Full Outer Join) 결과에서 나올 수 없는 투플을 찾는 문항이다.

릴레이션 R(A, B, C):
a1 b1 c1
a1 b1 c2
a3 b1 c2
a4 b2 c3

릴레이션 S(B, C, D):
b1 c1 d1
b1 c2 d2
b3 c3 d3

완전 외부 조인 (B, C 기준 자연 조인):
- (a1, b1, c1, d1): R(a1,b1,c1) ↔ S(b1,c1,d1) 매칭 → 생성 가능
- (a1, b1, c2, d2): R(a1,b1,c2) ↔ S(b1,c2,d2) 매칭 → 생성 가능
- (a3, b1, c2, d2): R(a3,b1,c2) ↔ S(b1,c2,d2) 매칭 → 생성 가능
- (a4, b2, c3, NULL): R(a4,b2,c3)에 매칭 S 없음 → D=NULL, 생성 가능
- (NULL, b3, c3, d3): S(b3,c3,d3)에 매칭 R 없음 → A=NULL, 생성 가능

각 보기 분석:
① a1 b1 c1 d1 — R(a1,b1,c1)과 S(b1,c1,d1)이 매칭 → 생성 가능.
② - b1 c2 d2 — A=NULL이려면 S(b1,c2,d2)에 매칭되는 R 투플이 없어야 한다. 그러나 R에 (a1,b1,c2)와 (a3,b1,c2)가 있어 S(b1,c2,d2)와 매칭됨 → A가 NULL인 투플이 생성될 수 없음. 정답.
③ a4 b2 c3 - — R(a4,b2,c3)에 매칭 S 없음 → D=NULL로 생성 가능.
④ - b3 c3 d3 — S(b3,c3,d3)에 매칭 R 없음 → A=NULL로 생성 가능.

핵심 정리:
완전 외부 조인: 내부 조인 결과 + 매칭 없는 R 투플(D=NULL) + 매칭 없는 S 투플(A=NULL).
NULL이 붙는 조건: 반대편 릴레이션에서 매칭 투플이 없을 때만.
(b1,c2)는 R에 매칭 투플이 2개 있으므로 ②처럼 A=NULL인 투플은 절대 생성되지 않는다.
