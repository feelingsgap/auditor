정답: ② (2번)

해설:
릴레이션 R(A, B, C, D, E, F)과 함수적 종속성을 기반으로 후보 키를 결정하는 문항이다.

주어진 함수적 종속성:
A → B, C → A, C → D, (A, F) → E

후보 키 도출 분석:
CF 검토: C → A (주어짐), A → B, C → D가 있으므로
  - CF → A (C → A)
  - CF → B (C → A → B)
  - CF → D (C → D)
  - CF → E (C → A이므로 (A,F)와 동치, (A,F) → E)
  - CF → C, F (자명)
  CF는 R의 모든 속성 {A,B,C,D,E,F}를 결정 → 슈퍼키.
  최소성 확인: C만으론 E 불가(F 필요), F만으론 A,B,C,D 불가 → CF는 후보 키.

나머지 보기 검토:
① AF: (A,F) → E, A → B. 그러나 A나 F 어느 쪽에서도 C나 D를 결정하는 FD 없음 → 슈퍼키 아님.
③ AC: A → B, C → D, C → A. 그러나 AC에서 E를 결정하려면 F가 필요하고, F에 대한 FD 없음 → 슈퍼키 아님.
④ AFC: CF가 이미 슈퍼키이므로 AFC는 슈퍼키이나 최소성 불만족 → 후보 키 아님.

각 보기 분석:
① AF — A와 F만으로는 C, D 결정 불가. 후보 키 아님.
② CF — 모든 속성 결정 가능 + 최소성 만족. 정답.
③ AC — E 결정에 F가 반드시 필요. 후보 키 아님.
④ AFC — CF가 이미 슈퍼키이므로 최소성 위반. 후보 키 아님.

핵심 정리:
후보 키: 릴레이션의 모든 속성을 결정하는 최소 슈퍼키.
이행적 종속성 추적: C → A, A → B → C는 B도 결정. FD 폐포(closure) 계산이 핵심.
(A,F) → E는 CF에서 C → A 이행 적용으로 (C,F) → E 도출 가능.
