정답: ③ (3번)

해설:
left=22, right=21 입력값으로 문장 커버리지(Statement Coverage)를 계산하는 문항이다. 각 라인을 하나의 문장으로 간주한다.

코드 실행 추적 (left=22, right=21):
라인 1: int ln = left;  → ln = 22  ✓ 실행
라인 2: int rn = right; → rn = 21  ✓ 실행
라인 3: if (ln > 21)    → 22 > 21 = true  ✓ 실행
라인 4:   ln = 0;       → ln = 0   ✓ 실행 (조건 true)
라인 5: if (rn > 21)    → 21 > 21 = false  ✓ 실행
라인 6:   rn = 0;       → 실행 안 됨 (조건 false)  ✗ 미실행
라인 7: if (ln > rn)    → 0 > 21 = false  ✓ 실행
라인 8:   return ln;    → 실행 안 됨 (조건 false)  ✗ 미실행
라인 9: else            → ✓ 실행 (단 가지 선택)
라인 10:  return rn;    → rn = 21 반환  ✓ 실행

결과: 10개 라인 중 8개 실행 → 문장 커버리지 = 8/10 = 80%

각 보기 분석:
① 60% — 6개 실행에 해당. 실제로는 8개이므로 오답.
② 70% — 7개 실행에 해당. 오답.
③ 80% — 8개 실행 / 10개 전체 = 80%. 정답.
④ 100% — 모든 라인이 실행되어야 하지만 6번(rn=0)과 8번(return ln)은 미실행. 오답.

핵심 정리:
문장 커버리지 = 실행된 문장 수 / 전체 문장 수 × 100%
미실행 라인: 라인 6(rn=0, rn이 21이므로 > 21 조건 불성립)과 라인 8(return ln, ln=0으로 rn=21보다 크지 않음).
문제 조건 "각 라인에 번호가 부여된 것 같이 문장으로 간주"에 따라 else(라인 9)도 1개 문장으로 카운트.
