정답: ④ (4번)

해설:
이진 분류 의사결정 트리에서 Gini Index와 분류 오류(Classification Error)를 계산하는 문항이다.

Gini Index 공식:
Gini = 1 - Σ p_i²
이진 분류에서 두 클래스의 비율이 p, (1-p)일 때: Gini = 1 - (p² + (1-p)²) = 2p(1-p)

분류 오류(Classification Error):
Error = 1 - max(p_i)

각 보기 분석:
① 각 클래스 0개, 6개: p1=0, p2=1
   Gini = 1 - (0² + 1²) = 1 - 1 = 0.
   완전 순수한 노드의 Gini=0. 맞는 설명.

② 각 클래스 1개, 5개: p1=1/6, p2=5/6
   Gini = 1 - ((1/6)² + (5/6)²) = 1 - (1/36 + 25/36) = 1 - 26/36 = 10/36 ≈ 0.278 > 0.
   Gini가 0보다 크다는 설명. 맞는 설명.

③ 각 클래스 3개, 3개: p1=0.5, p2=0.5
   분류 오류 = 1 - max(0.5, 0.5) = 1 - 0.5 = 0.5.
   분류 오류 비율이 0.5. 맞는 설명.

④ 각 클래스 3개, 3개: p1=0.5, p2=0.5
   Gini = 1 - (0.5² + 0.5²) = 1 - (0.25 + 0.25) = 1 - 0.5 = 0.5.
   그러나 "Gini Index는 1이다"라고 설명. 실제 값은 0.5이므로 틀린 설명. 정답.

핵심 정리:
Gini Index 범위: 0(완전 순수) ~ 0.5(이진 분류의 최대 불순도, 50:50일 때).
Gini는 절대 1이 될 수 없음. 이진 분류에서 최댓값은 0.5.
분류 오류와 Gini는 다른 불순도 측도: 분류 오류는 최대 0.5, Gini도 이진 분류에서 최대 0.5.
