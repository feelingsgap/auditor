정답: ③ (3번)

해설:
공공부문 초거대 AI 도입·활용 가이드라인(2024.04.)에서 LLM 사용 방식을 파인튜닝으로 결정했을 때의 체크리스트를 묻는 문항이다.

각 보기 분석:
① 업무 특성을 반영한 LLM 선정 여부 — 파인튜닝에 사용할 기반 모델(Foundation Model) 선정은 파인튜닝의 첫 번째 단계로, 적절한 체크리스트 항목이다.
② 추가학습을 위한 데이터 준비 여부 — 파인튜닝은 도메인 특화 데이터로 모델을 추가 학습하는 방식이므로, 학습 데이터 준비 여부는 핵심 체크항목이다.
③ 파운데이션 모델 고도화 및 기대 성능 모니터링 검토 — 부적절한 항목이다. 파운데이션 모델 자체를 고도화(Foundation Model 개선)하는 것은 파인튜닝이 아닌 사전학습(Pre-training) 또는 모델 개발 영역에 해당한다. 파인튜닝 사용 결정 후 체크리스트에서는 기반 모델 고도화가 아닌 파인튜닝 적용 방법·데이터·성능 검증에 집중한다.
④ 모델 버전 관리 및 배포 방안 마련 검토 — 파인튜닝 완료 후 모델의 버전 관리와 운영 배포 방안은 파인튜닝 적용 시 고려할 적절한 체크항목이다.

핵심 정리:
파인튜닝(Fine-tuning): 기존 파운데이션 모델을 도메인 특화 데이터로 추가 학습.
체크리스트 핵심: 기반 모델 선정 + 학습 데이터 준비 + 버전·배포 관리.
파운데이션 모델 고도화는 파인튜닝 범위를 초과하는 활동 (Pre-training 영역).
