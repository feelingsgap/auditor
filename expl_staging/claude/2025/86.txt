정답: ④ (4번)

해설:
가상 기억장치(Virtual Memory)에 대한 설명 중 옳지 않은 것을 찾는 문항이다.

각 보기 분석:
① "CPU가 참조하는 가상주소를 주기억 장치의 실제 주소로 변환하는 주소 매핑이 필요하다."
   가상 주소 → 물리 주소 변환은 MMU(Memory Management Unit)가 담당. 맞는 설명.

② "프로그램이 적재할 주기억 장치의 주소를 물리주소라고 한다."
   물리주소(Physical Address) = 실제 메모리(RAM) 주소. 맞는 설명.

③ "가상주소가 주기억 장치에 존재하지 않는 경우에는 페이지 부재(fault)가 발생한다."
   페이지 부재(Page Fault): 요청한 페이지가 메인 메모리에 없을 때 발생하여 보조기억장치에서 가져옴. 맞는 설명.

④ "LFU는 최근까지 가장 오랫동안 사용하지 않은 페이지를 선택하여 교체한다."
   틀린 설명. 정의 혼동:
   - LFU(Least Frequently Used): 사용 빈도가 가장 낮은(사용 횟수가 가장 적은) 페이지 교체.
   - LRU(Least Recently Used): 가장 오랫동안 사용하지 않은(가장 오래 전에 마지막으로 사용된) 페이지 교체.
   ④는 LRU의 설명을 LFU로 잘못 기술. 정답.

핵심 정리:
페이지 교체 알고리즘:
LRU(Least Recently Used): 가장 오래 전에 사용한 페이지 교체 → 스택, 시간 기록 사용.
LFU(Least Frequently Used): 사용 횟수가 가장 적은 페이지 교체 → 카운터 사용.
FIFO: 가장 먼저 적재된 페이지 교체.
OPT: 앞으로 가장 오랫동안 사용하지 않을 페이지 교체 (이론적 최적).
