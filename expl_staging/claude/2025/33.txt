정답: ② (2번)

해설:
AI 테스트 기법의 설명과 기법명의 연결 중 부적절한 것을 묻는 문항이다.

각 항목 분석:
① 가 - 메타모픽 테스트 기법(metamorphic testing) — 적절한 연결이다. 메타모픽 테스트는 입력 데이터를 변환했을 때 출력이 예상된 관계(메타모픽 관계)를 만족하는지 검증한다. "입력 변화에 대한 추론 결과의 관련성"이 핵심이다.
② 나 - 커버리지 테스트 기법(coverage testing) — 부적절한 연결이다. 나의 설명("타당한 추론 결과를 얻을 수 있는 정확한 범위를 파악하지 못해도 타당한 값을 얻을 수 있는 예상되는 범위가 있다")은 커버리지 테스트가 아니라 AI의 강건성(Robustness) 또는 메타모픽 테스트의 신뢰 영역 개념에 가깝다. 커버리지 테스트는 특정 커버리지 기준(문장, 분기 등)을 달성하도록 테스트 케이스를 설계하는 기법으로, 이 설명과 맞지 않는다.
③ 다 - 뉴런 커버리지 테스트 기법(neuron coverage testing) — 적절한 연결이다. 뉴런 커버리지 테스트는 신경망의 뉴런 활성화 여부를 측정하고 활성화되지 않은 뉴런을 커버하도록 테스트 데이터를 추가하는 기법이다.
④ 라 - 적대적 공격 기법(adversarial attack) — 적절한 연결이다. 적대적 공격은 인위적으로 노이즈를 추가한 입력(adversarial example)으로 AI 모델이 잘못된 결과를 내도록 유도하는 기법이다.

핵심 정리:
메타모픽 테스트: 입력 변환 → 출력 관계 검증.
뉴런 커버리지 테스트: 비활성 뉴런 탐지 → 테스트 데이터 보완.
적대적 공격: 의도적 노이즈 입력 → 모델 오동작 유도.
커버리지 테스트: 코드의 실행 경로 커버율 측정 — 나의 설명과 다름.
