정답: ③ (3번)

해설:
「정보시스템 하드웨어 규모 산정 지침(TTAK.KO-10.0292/R3)」에서 WEB/WAS 산정의 연계 부하 보정 기준을 묻는 문항이다.

연계 부하 보정 기준 (지침 기준):
- WEB 서버: 연계 부하 보정 미적용 (WEB은 정적 콘텐츠 처리 위주, 연계 업무 부담 없음)
- WAS 서버:
  - 연계 업무 없음: 보정치 미적용 (0%)
  - 연계 업무가 단순 조회만 있음: 전체 부하의 10% 추가
  - 연계 업무에 대내/대외 갱신 업무 포함: 전체 부하의 30% 추가

각 보기 분석:
① WEB 서버의 경우에는 추가적인 보정치를 적용하지 않는다. — WEB 서버 연계 보정 미적용. 맞는 설명.
② WAS 서버는 전체 업무 중 연계 업무가 없는 경우에는 추가적인 보정치를 적용하지 않는다. — 연계 없으면 보정 없음. 맞는 설명.
③ WAS 서버는 전체 업무 중 연계 업무가 단순 조회 업무만 있는 경우에 전체 부하의 20%를 추가 적용한다. — 단순 조회 연계 시 10% 적용이 올바름. 20%는 잘못된 수치. 틀린 설명. 정답.
④ WAS 서버 전체 업무 중 연계 업무가 대내/대외 갱신 업무가 있는 경우에 전체 부하의 30%를 추가 적용한다. — 갱신 연계 시 30% 적용. 맞는 설명.

핵심 정리:
WEB/WAS 연계 부하 보정 수치:
단순 조회 연계 → +10%
대내/대외 갱신 연계 → +30%
WEB 서버 → 보정 없음
WAS 서버 연계 없음 → 보정 없음
