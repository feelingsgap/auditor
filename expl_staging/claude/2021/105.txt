정답: ④ (4번)

해설:
무선 통신 암호화 기술인 WPA2와 WPA-EAP의 특성을 묻는 문항이다. WPA-EAP는 세션별로 동적 키를 사용한다는 점을 이해해야 한다.

각 보기 분석:
① WPA2는 CCMP 알고리즘을 사용하여 암호화한다 — WPA2는 AES 기반의 CCMP(Counter Mode with Cipher Block Chaining Message Authentication Code Protocol)를 사용한다. 올바른 설명이다.
② WPA2는 Access Point에 접속하는 단말기마다 지정된 암호화 키를 재사용한다 — WPA2-Personal(PSK) 방식에서는 모든 단말이 동일한 사전 공유 키를 사용하므로 올바른 설명이다.
③ WPA-EAP는 사용자가 입력하는 아이디, 패스워드를 사용하는 사용자 인증 방식이다 — WPA-EAP는 802.1X 기반 인증으로 RADIUS 서버를 통해 사용자 인증을 수행한다. 올바른 설명이다.
④ WPA-EAP는 사용자별 무선랜 연결 세션마다 지정된 암호화 키를 재사용한다 — WPA-EAP(WPA-Enterprise)는 세션마다 동적으로 생성된 고유한 암호화 키를 사용하며, 동일 키를 재사용하지 않는다. 잘못된 설명이다. 정답.

핵심 정리:
WPA2-Personal(PSK): 모든 단말이 동일한 사전 공유 키 사용.
WPA2-Enterprise(EAP): 802.1X + RADIUS 인증, 세션마다 동적 키 생성.
WPA-EAP의 핵심 보안 장점: 사용자별·세션별 동적 키 사용으로 키 재사용 위험 제거.
