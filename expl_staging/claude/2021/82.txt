정답: ② (2번)

해설:
주 7일(월~금) 24시간 운영 시스템에서 MTTR(Mean Time To Repair, 평균 수리 시간)을 구하는 문항이다.
MTTR = 총 장애 시간 / 장애 발생 횟수

장애 발생 내역:
- 월요일: 13:00~14:00 → 1시간
- 화요일: 장애 없음
- 수요일: 장애 없음
- 목요일: 16:00~19:00 → 3시간
- 금요일: 장애 없음

총 장애 시간 = 1시간 + 3시간 = 4시간
장애 발생 횟수 = 2회

MTTR = 4시간 / 2회 = 2시간

각 보기 분석:
① 1시간 — 장애 횟수를 고려하지 않은 오답.
② 2시간 — (1+3)/2 = 2시간. 정답.
③ 3시간 — 오답.
④ 4시간 — 총 장애 시간이지 MTTR이 아님.

핵심 정리:
MTTR(Mean Time To Repair): 평균 수리 시간 = 총 장애 시간 / 장애 횟수.
MTTF(Mean Time To Failure): 평균 고장 시간 = 정상 운영 시간 / 고장 횟수.
MTBF(Mean Time Between Failures) = MTTF + MTTR.
가용성(Availability) = MTTF / MTBF = MTTF / (MTTF + MTTR).
