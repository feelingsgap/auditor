정답: ① (1번)

해설:
빅데이터 아키텍처에서 수집 레이어에 해당하지 않는 기술을 찾는 문항이다.
빅데이터 아키텍처는 수집, 적재, 처리 및 탐색, 분석 및 응용의 6개 레이어로 구성된다.

각 보기 분석:
① 머하웃(Mahout) — Apache Mahout은 Hadoop 기반의 분산 머신러닝 라이브러리이다. 분류, 군집화, 추천 알고리즘을 제공하는 분석 및 응용 레이어의 기술이지, 수집 레이어 기술이 아니다. 정답.
② 스톰(Storm) — Apache Storm은 실시간 스트리밍 데이터 처리 프레임워크이다. 실시간으로 데이터를 수집하고 처리하는 수집/처리 레이어 기술이다.
③ 카프카(Kafka) — Apache Kafka는 분산 메시지 큐/이벤트 스트리밍 플랫폼으로, 대용량 데이터를 실시간으로 수집하고 전달하는 대표적인 수집 레이어 기술이다.
④ 플럼(Flume) — Apache Flume은 로그 데이터 등 대용량 데이터를 안정적으로 수집, 집계, 이동시키는 수집 레이어 기술이다.

핵심 정리:
빅데이터 수집 레이어 기술: Kafka, Flume, Storm, Sqoop, Logstash 등.
분석/응용 레이어 기술: Mahout(머신러닝), Spark MLlib, R, Python 등.
Mahout: Hadoop 기반 분산 머신러닝, 협업 필터링·분류·군집화 알고리즘 제공.
