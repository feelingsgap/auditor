정답: ② (2번)

해설:
컨볼루션 신경망(CNN)에서 5×5 입력 이미지에 3×3 필터를 스트라이드(stride) 1로 합성곱 연산을 적용할 때 생성되는 특징 맵(feature map)의 크기를 구하는 문항이다.
패딩은 적용하지 않는다(zero padding 없음).

특징 맵 크기 공식:
출력 크기 = (입력 크기 - 필터 크기) / 스트라이드 + 1

계산:
- 입력: 5×5
- 필터: 3×3
- 스트라이드: 1
- 패딩: 없음(0)

출력 가로 = (5 - 3) / 1 + 1 = 2 + 1 = 3
출력 세로 = (5 - 3) / 1 + 1 = 2 + 1 = 3
→ 특징 맵 크기 = 3×3

각 보기 분석:
① 2×2 — 오답.
② 3×3 — (5-3)/1+1 = 3. 정답.
③ 4×4 — 오답.
④ 5×5 — 패딩이 있을 때 same 패딩 결과이지만 문제에서 패딩 없음을 명시. 오답.

핵심 정리:
CNN 출력 크기: ((N - F + 2P) / S) + 1 (N=입력, F=필터, P=패딩, S=스트라이드).
패딩 없는 경우: 출력 크기 < 입력 크기 (정보 손실 발생).
Same 패딩: 출력 크기 = 입력 크기 (패딩으로 경계 보완).
