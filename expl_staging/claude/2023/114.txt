정답: ② (2번)

해설:
웹 취약점 진단 도구는 웹 애플리케이션의 보안 취약점(SQL 인젝션, XSS, 세션 취약점 등)을 탐지하기 위한 프록시·스캐너 기반 도구를 의미한다. 이와 가장 거리가 먼 것을 찾아야 한다.

각 보기 분석:
① Burp Suite — 웹 애플리케이션 보안 테스트를 위한 통합 플랫폼이다. HTTP 프록시, 스파이더, 스캐너, 인트루더 등 다양한 웹 취약점 진단 기능을 제공한다. 웹 취약점 진단 도구가 맞다.
② Hydra — 다양한 프로토콜(FTP, SSH, HTTP, 등)에 대한 패스워드 브루트 포스(brute force) 공격 도구이다. 웹 취약점 진단 전용 도구가 아니라 계정 탈취 공격 도구이다. 정답.
③ Paros — Java 기반 HTTP/HTTPS 프록시로, 웹 애플리케이션 취약점 진단에 사용되는 도구이다. 현재는 Paros를 기반으로 한 OWASP ZAP으로 발전하였다.
④ WebScarab — OWASP에서 개발한 웹 애플리케이션 취약점 분석 프레임워크로, HTTP 프록시 기반의 웹 보안 테스트 도구이다.

핵심 정리:
웹 취약점 진단 도구: Burp Suite, OWASP ZAP, Paros, WebScarab, Nikto, Acunetix.
Hydra: 네트워크 서비스 대상 패스워드 공격(브루트 포스·사전 공격) 도구로 분류가 다르다.
OWASP Top 10 취약점 진단 시 Burp Suite, ZAP이 가장 널리 사용된다.
