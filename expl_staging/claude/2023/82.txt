정답: ④ (4번)

해설:
C 프로그램에서 char C = '9'로 선언 시 문자 변수 C의 메모리 값을 16진수로 구하는 문제이다. 단, ASCII Code에서 문자 '0'은 0011 0000(0x30)임이 주어진다.

ASCII 코드 계산:
- '0'의 ASCII 값: 0x30 = 48(십진수)
- 문자 '9'는 '0'부터 9번째 문자이므로: 0x30 + 9 = 0x39
- 또는 이진수: 0011 0000 + 0000 1001 = 0011 1001 = 0x39

각 보기 분석:
① 0x09 — 0x09는 수평 탭(HT) 제어 문자의 ASCII 값. '9' 문자가 아님.
② 0x19 — 해당 값은 EM(End of Medium) 제어 문자. '9' 문자 아님.
③ 0x29 — '0'(0x30)보다 작으므로 숫자 문자 범위 밖.
④ 0x39 — '0'(0x30) + 9 = 0x39. 문자 '9'의 ASCII 값. 정답.

핵심 정리:
ASCII 숫자 문자 범위: '0'(0x30) ~ '9'(0x39).
문자 'n'의 ASCII 값 = 0x30 + n (n은 0~9).
char 형 변수에 문자 리터럴을 대입하면 해당 문자의 ASCII 코드 값이 저장된다.
숫자 정수 9와 문자 '9'(0x39)는 다른 값이다.
