정답: ④ (4번)

해설:
서비스 거부(DoS) 공격의 유형별 특징을 구별하는 문항이다.

각 보기 분석:
① TearDrop 공격은 IP 패킷의 시퀀스 넘버가 서로 중첩되도록 조작하여 수신측의 IP 패킷 재조합 시 오류를 발생시키는 공격이다 — 적절한 설명이다. 분할(fragment) 패킷의 오프셋 값을 중첩되도록 조작하여 재조합 시 버퍼 오버플로나 시스템 크래시를 유발한다.
② TCP SYN 플러딩 공격은 TCP의 연결 과정인 3-Way Handshaking의 문제점을 악용한 공격이다 — 적절한 설명이다. SYN 패킷을 대량 전송하여 서버의 연결 대기 큐를 고갈시키는 공격이다.
③ Smurf 공격은 브로드캐스팅 방식으로 동작할 수 있는 ICMP 프로토콜의 특성을 악용한 공격이다 — 적절한 설명이다. 출발지 IP를 피해자 IP로 위조하여 브로드캐스트 주소로 ICMP Echo Request를 전송, 대량의 응답이 피해자에게 집중된다.
④ Bonk 공격은 패킷을 전송할 때 IP 주소와 목적지 IP 주소 값을 동일하게 작성하여 공격 대상에게 보내는 공격이다 — 부적절한 설명이다. 소스 IP와 목적지 IP를 동일하게 작성하는 것은 Land 공격이다. Bonk 공격은 TearDrop과 유사하게 UDP 패킷의 시퀀스 번호를 비정상적으로 조작하여 수신측 재조합 오류를 유발하는 DoS 공격이다. 정답.

핵심 정리:
Land 공격: 소스 IP = 목적지 IP (동일 주소 전송).
Bonk/Boink 공격: 패킷 시퀀스 번호 조작으로 재조합 오류 유발 (TearDrop 변형).
TearDrop: IP 단편화 오프셋 중첩 조작.
