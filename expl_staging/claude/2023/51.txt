정답: ③ (3번)

해설:
ISO 8000은 데이터 품질에 관한 국제 표준이다. 괄호 안에 들어갈 용어는 "데이터가 표준 코드 값 또는 표준 도메인 값에 위배될 경우 발생하는 오류"에 해당하는 데이터 품질 기준이다.

각 보기 분석:
① 충분성(Sufficiency) — 필요한 데이터가 충분히 존재하는지에 관한 품질 기준이다. 코드/도메인 위배와 무관하다.
② 사실성(Factuality/Accuracy) — 데이터가 실제 세계를 정확하게 반영하는지에 관한 기준이다. 코드 값 표준 위배와는 다른 개념이다.
③ 적합성(Conformity) — 데이터가 표준 코드 값, 표준 도메인 값, 또는 정해진 형식과 일치하는지에 관한 품질 기준이다. 표준 코드 값 또는 표준 도메인 값에 위배될 경우 발생하는 오류는 적합성 오류에 해당한다. 정답.
④ 필수성(Completeness) — 필수적으로 값이 있어야 하는 데이터 항목에 값이 존재하는지에 관한 기준이다. NULL 여부와 관련된다.

핵심 정리:
ISO 8000 데이터 품질 기준 주요 유형:
- 적합성(Conformity): 표준 코드/도메인/형식 준수 여부.
- 정확성(Accuracy): 실제 값과의 일치 여부.
- 완전성(Completeness): 필수 데이터의 존재 여부.
- 일관성(Consistency): 데이터 간 상호 모순 없음.
