정답: ③ (3번)

해설:
멀티미디어 데이터 검색 기법 중 "수학 함수로 저장된 데이터와 질의 조건으로 주어진 데이터 간의 유사도를 수학 함수로 계산하여 유사도가 높은 데이터를 검색한다"는 설명에 해당하는 기법을 묻는 문제이다.

설명 분석:
- 수학 함수로 데이터를 저장(특징 벡터 등으로 표현)
- 질의 데이터와 저장 데이터 간 유사도를 수학 함수(거리 함수, 유사도 함수)로 계산
- 유사도 기반 검색 수행
→ 이는 매칭 기법(matching technique)의 핵심 개념

각 보기 분석:
① 랭킹 기법(ranking technique) — 검색 결과를 관련도 순으로 정렬하는 기법으로, 유사도 계산 자체가 아닌 결과 정렬에 초점.
② 인덱스 기법(index technique) — 빠른 검색을 위해 색인을 구축하는 기법으로, 유사도 계산과는 별개의 개념.
③ 매칭 기법(matching technique) — 데이터를 수학적 표현으로 변환한 후 질의와의 유사도를 수학 함수로 계산하여 일치하는 데이터를 찾는 기법. 설명과 정확히 일치. 정답.
④ 필터링 기법(filtering technique) — 조건에 맞지 않는 데이터를 걸러내는 방식으로, 유사도 계산 중심이 아님.

핵심 정리:
매칭 기법: 특징 벡터 추출 → 유사도 함수(유클리드 거리, 코사인 유사도 등) 계산 → 임계값 이상 결과 반환.
멀티미디어 검색에서 이미지, 음성, 영상 데이터를 수치 벡터로 변환하여 매칭에 활용한다.
