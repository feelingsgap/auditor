정답: ② (2번)

해설:
릴레이션 A(P, Q, R)와 B(Q, R, S)에 대해 결과 릴레이션 C(P, Q, R, S)가 나오도록 하는 관계 대수 연산을 찾는 문항이다. C 결과를 보면 p3-q1-r2에 대해 B에 매칭되는 튜플이 없음에도 NULL로 포함되어 있다.

결과 C 분석:
- p1, q1, r1: B에 (q1, r1, s1), (q1, r1, s2)가 있으므로 2개 행 생성
- p2, q1, r1: 동일하게 2개 행 생성
- p3, q1, r2: B에 (q1, r2)가 없으므로 NULL 포함 → 외부 조인
- p4, q2, r3: B에 (q2, r3, s3)가 있으므로 1개 행 생성

p3의 S 값이 NULL인 것은 A에 있지만 B에 매칭이 없어도 A의 튜플이 유지되는 Left Outer Join(좌외부 조인) 특성이다.

각 보기 분석:
① A ⋈N B (자연 조인) — 매칭되지 않는 p3 행이 결과에서 제외된다. C와 불일치.
② A ⋈+ B (외부 조인, Left Outer Join) — A의 모든 튜플이 포함되고 매칭 없는 경우 NULL로 채워진다. C와 일치. 정답.
③ A × B (세미 조인) — A에서 B와 매칭되는 튜플만 A의 속성으로 반환. S 속성이 포함되지 않는다.
④ A U+ B (외부 합집합) — 속성이 다른 두 릴레이션의 합집합으로 C의 구조와 다르다.

핵심 정리:
Left Outer Join: 왼쪽 릴레이션의 모든 튜플 유지, 오른쪽 매칭 없는 경우 NULL 채움.
자연 조인: 매칭되는 튜플만 결과에 포함.
외부 조인은 매칭 실패 시 NULL 허용으로 데이터 손실을 방지한다.
