정답: ③ (3번)

해설:
RAID(Redundant Array of Independent Disks)의 각 레벨별 특성에 대한 설명 중 옳지 않은 것을 찾는 문항이다.

RAID 레벨별 특성:
- RAID-0: 스트라이핑만 사용. 성능 극대화, 결함 허용 없음.
- RAID-1: 미러링(복제). 안정성 극대화, 용량 효율 50%.
- RAID-2: 해밍코드를 오류 수정 코드로 사용. (사실상 사용 안 됨)
- RAID-3: 비트 또는 바이트 단위 스트라이핑 + 전용 패리티 디스크(패리티 코드 사용).
- RAID-4: 블록 단위 스트라이핑 + 전용 패리티 디스크.
- RAID-5: 블록 단위 스트라이핑 + 패리티를 모든 디스크에 분산.
- RAID-6: RAID-5 + 이중 패리티 (동시에 2개 디스크 장애 허용).

각 보기 분석:
① RAID-0은 성능을 추구하고 RAID-1은 안정성을 추구한 방식이다 — 정확한 설명이다. 옳다.
② RAID-2와 RAID-3은 성능향상을 위해 실제 데이터를 저장하는 디스크를 RAID-0으로 구성한 방식이다 — RAID-2와 RAID-3은 데이터 디스크에 스트라이핑(RAID-0 방식)을 적용하고 별도 오류 정정 코드/패리티 디스크를 추가한다. 옳다.
③ RAID-3은 데이터 복구를 위해 해밍코드를 사용하고, RAID-4는 패리티 코드를 이용한다 — RAID-3은 해밍코드가 아닌 패리티 코드(XOR 기반)를 사용한다. 해밍코드를 사용하는 것은 RAID-2이다. 틀린 설명이다. 정답.
④ RAID-5와 RAID-6은 데이터 복구를 위해 다른 디스크에 패리티 정보를 함께 저장해 안정성을 높인 방식이다 — 패리티를 분산 저장하는 RAID-5와 이중 패리티의 RAID-6에 대한 정확한 설명이다. 옳다.

핵심 정리:
RAID-2: 해밍코드(ECC) 사용 → 현재 거의 사용 안 함.
RAID-3: XOR 패리티 코드 + 전용 패리티 디스크 (비트/바이트 스트라이핑).
RAID-4: XOR 패리티 코드 + 전용 패리티 디스크 (블록 스트라이핑).
RAID-5: 분산 패리티. RAID-6: 이중 분산 패리티.
