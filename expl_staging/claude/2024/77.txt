정답: ③ (3번)

해설:
디지털 논리회로 중 다수의 입력 단자에서 조건에 맞는 하나를 선택하여 단일 출력 단자로 연결하는 조합회로를 찾는 문항이다.

문제 조건: "다수의 입력 단자 중에서 조건에 맞는 하나를 선택하여 단일 출력 단자로 연결하는 조합회로"

각 보기 분석:
① 인코더(Encoder) — 2^n개의 입력 중 하나가 활성화되면 n비트 이진 코드로 변환하는 회로. 다수 입력을 압축된 코드로 출력한다. 선택하여 연결하는 기능이 아니다. 틀렸다.
② 디코더(Decoder) — n비트 입력을 받아 2^n개의 출력 중 하나를 활성화하는 회로. 입력이 하나이고 출력이 다수다. 반대 방향이다. 틀렸다.
③ 멀티플렉서(Multiplexer, MUX) — 2^n개의 데이터 입력 중 n비트 선택 신호에 따라 하나를 선택하여 단일 출력으로 내보내는 회로. "다수 입력 → 선택 → 단일 출력" 구조와 정확히 일치한다. 정답.
④ 디멀티플렉서(Demultiplexer, DEMUX) — 단일 입력을 선택 신호에 따라 여러 출력 중 하나로 보내는 회로. 멀티플렉서와 반대 방향이다. 틀렸다.

핵심 정리:
멀티플렉서(MUX): 다수 입력 → 1개 출력. "많은 것 중 선택" = 데이터 선택기(data selector).
디멀티플렉서(DEMUX): 1개 입력 → 다수 출력. "하나를 여럿으로 분배".
디코더: n비트 입력 → 2^n개 출력(하나 활성화). 인코더: 2^n개 입력 → n비트 출력.
MUX는 선택 신호(select line)로 입력 채널을 지정한다.
